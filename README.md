# 🎵 Spotify Clone

A Spotify-inspired music streaming web application built using **React + Vite**.  
This project replicates the core UI and functionality of Spotify including playlists, music player controls, and responsive design.

## 🚀 Features
- Play / Pause / Next / Previous controls
- Song playlist UI
- Responsive design for mobile & desktop
- Interactive music progress bar
- Clean Spotify-like UI

## 🛠️ Tech Stack
- React.js
- Vite
- JavaScript (ES6)
- HTML5 & CSS3 / Tailwind
- Node.js (for future backend)

## 🎯 Purpose
This project was created to practice **React concepts**, UI cloning, and real-world frontend development skills.

## 📌 Future Improvements
- Add backend using MERN stack
- User login/signup
- Create custom playlists
- Add search functionality
- Connect Spotify API
